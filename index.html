Développement d' un algorithme de statistiques en Javascript

Dans le cadre de mon stage de fin de DUT chez la start-up Octipas, j'avais pour mission de développer un algorithme permettant de récupérer les données des terminaux de vente fournis aux magasins par Octipas et d'en tirer des statistiques. Le but était ainsi de savoir quels magasins utilisaient le plus les terminaux, lesquels étaient les plus performants, etc.

Ce programme, développé en Javascript, utilise un algorithme de Machine Learning ( Algorithme des k-moyennes ) afin de calculer et d'afficher ces statistiques.

Technologies utilisées: Javascript, HTML, CSS, PHP

Migration d'un site web vers un nouveau framework PHP

Dans le cadre de mon stage de fin de DUT chez la start-up Octipas, j'ai également eu pour mission de commencer la migration du site de listes de cadeaux Ookoodoo ( 250 000 utilisateurs ), depuis le framework PHP Drupal 6 vers un nouveau, nommé Laravel.

J'ai ainsi pu voir le développement d'un site web en PHP à l'aide d'un framework, ainsi que les outils de localisation de Laravel, qui permettent d'afficher un contenu différent selon la langue de l'utilisateur.

Technologies utilisées: PHP ( Laravel ), HTML, CSS, Git

Création et développement d'un framework front-end

Durant mon temps libre, j'ai décidé de créer un framework front-end, sous la forme d'une librairie CSS et de fichiers Javascript, dont le but est d'accélérer le processus de mise en page d'un site web ( dans la lignée de Bootstrap ).

Le code source est disponible sur GitHub, et la librairie représente actuellement plus de 2500 lignes de CSS, et 500 lignes de Javascript.

Technologies utilisées: HTML, CSS ( SASS ), Javascript, Git
